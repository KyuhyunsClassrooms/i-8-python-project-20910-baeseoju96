# 실행 결과 기록

최종 프로그램을 실행한 뒤, 터미널에 나온 결과를 아래에 복사하세요.
========================================
AI 활용 자유 주제 파이썬 미니 프로젝트
주제: 다기준 의사결정(MCDA) 기반 학과 선택기
========================================
자신의 진로 기준별 중요도(1~5점)를 입력하면
알고리즘을 통해 최적의 학과를 추천해 드립니다.

=== 진로 선택 가중치 입력 (1점 ~ 5점) ===
1. 나의 '적성'은 얼마나 중요한가요? : 5
2. 학과의 '발전가능성'은 얼마나 중요한가요? : 4
3. 나의 '흥미'는 얼마나 중요한가요? : 2

================================================
           최종 학과 분석 결과          
================================================
▶ 경제학과 : 90점 [우수 추천]
▶ 컴퓨터공학과 : 97점 [우수 추천]
▶ 미디어학과 : 85점 [우수 추천]
▶ 심리학과 : 74점
▶ 정치외교학과 : 77점
================================================
```

실행할 때 사용한 입력값: 
1. 나의 '적성' 중요도: 5
2. 학과의 '발전가능성' 중요도: 4
3. 나의 '흥미' 중요도: 2

결과가 내가 의도한 대로 나왔는지 설명:
의도한 대로 정확하게 연산되어 출력되었습니다. 
프로그램 내부 2차원 리스트에 고정되어 있던 '컴퓨터공학과'의 고유 데이터인 적성(9점), 발전가능성(10점), 흥미(6점)에 
내가 입력한 가중치인 5, 4, 2가 각각 올바르게 곱해졌습니다.
그 결과 (9*5) + (10*4) + (6*2) = 45 + 40 + 12 = 97점이라는 종합 점수가 정확하게 산출되었습니다.
또한, 점수 출력 시 if 조건문 시스템이 정상 작동하여 80점 이상 100점 미만인 학과들(경제학과, 컴퓨터공학과, 미디어학과) 
옆에는 기획했던 대로 '[우수 추천]'이라는 등급 문구가 누락 없이 올바르게 표시된 것을 확인했습니다.
