## 3. 입력-처리-출력 구조

### 입력

사용자가 입력하거나 프로그램이 사용할 데이터는 무엇인가?

- 사용자가 입력하는 '적성 중요도' (1~5점)

-사용자가 입력하는 '발전가능성 중요도' (1~5점)

-사용자가 입력하는 '흥미 중요도' (1~5점)

-프로그램 내부에 미리 저장된 학과별 특성 데이터 (2차원 리스트)

### 처리

입력된 데이터를 어떤 규칙이나 알고리즘으로 처리할 것인가?

- 사용자가 입력한 문자열 데이터를 int()를 사용하여 정수형 데이터로 변환한다.

-2차원 리스트의 각 행(학과별 데이터)을 반복문(for)으로 하나씩 순회하며 꺼낸다.

-다기준 의사결정(MCDA) 공식에 따라 (학과별 기준 점수 * 사용자 가중치)를 각각 곱한 후, 이 값들을 모두 더해 학과별 총점을 산출한다. 

### 출력

프로그램이 최종적으로 무엇을 출력하는가?

- 각 학과의 이름과 계산된 종합 점수 결과
- 조건문(if)을 활용하여 총점 기준에 따라 분류된 맞춤형 추천 등급 표시 ([★ 강력 추천!], [우수 추천] 등)
